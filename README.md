**Prescripto HSP** is a role-based hospital management system for Doctors, Admins, and Patients with online appointment booking and Stripe payment integration.

- 🔐 **Secure Auth**: Google OAuth via Firebase + JWT-based backend access control  
- 📅 **Appointments**: Book, manage, and track medical appointments  
- 🧑‍⚕️ **Doctor Dashboard**: View patients, prescribe, manage visits  
- 🛡️ **Admin Panel**: Control users, roles, and hospital data  
- 💳 **Stripe Integration**: Secure payment processing for services  
- 🖼️ **Image Uploads**: Firebase storage for profile images & documents  
- 📱 **Responsive UI**: Built with Tailwind CSS + Redux Toolkit  

**Tech Stack**: React, Node.js, Express, MongoDB, Firebase, Tailwind CSS, Stripe API, Redux Toolkit, JWT

**RediStash** is a high-performance chat application built with real-time messaging, media sharing, and modern design aesthetics.

- 🔴 **Real-Time Messaging**: Built using Redis (Upstash) and Pusher  
- 🔐 **User Authentication**: Secure, role-based auth flow  
- ☁️ **Media Uploads**: Upload & share images via Cloudinary  
- ⚛️ **Smooth UX**: Built with Next.js 14 + App Router & Framer Motion  
- 🎨 **Modern UI**: Powered by Shadcn UI, TailwindCSS & Lucide Icons  
- 🔊 **Sound Effects**: Interactive UI with use-sound  
- 😊 **Emoji Support**: Integrated Emoji Mart  
- ⚙️ **Efficient State**: Zustand + TanStack Query for seamless UX  

**Tech Stack**: Next.js 14, TypeScript, Redis (Upstash), Pusher, Cloudinary, Zustand, Tailwind CSS, Shadcn UI, Framer Motion, TanStack Query, Lucide React, Emoji Mart, use-sound
